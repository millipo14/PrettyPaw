#  PrettyPaw - лендинг зоомагазина (Frontend)

Учебный проект одностраничного сайта зоомагазина с адаптивной версткой и базовой интерактивностью на JavaScript.


## Технологии

- HTML5
- SCSS
- JavaScript (ES6+)
- Webpack

## Описание проекта

Проект выполнен в рамках практики фронтенд-разработки. Основной фокус был на:

- верстке по макету
- адаптивном дизайне
- базовой логике на JavaScript
- работе с модульной сборкой (Webpack)


## Реализованный функционал

- Адаптивная верстка под разные устройства (desktop / tablet / mobile)
- Семантическая HTML-структура
- Стилизация интерфейса с использованием CSS/SCSS
- Интерактивные элементы на JavaScript
- Организация модульной структуры проекта
- Сборка проекта с помощью Webpack


## Примечание

Проект является учебной практикой и не представляет собой коммерческое приложение.
